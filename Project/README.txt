README.txt du Projet ChatOS - Tom REDON - M1 Informatique - Groupe 2

Veuillez vérifier que vous possédiez une version 11 de Java ou
bien une version ultérieure.

COMMANDES :
Pour lancer le serveur : java -jar ChatOSServer-1.0.numBuild port

Pour lancer le client : java -jar ChatOSClient-1.0.numBuild nomClient localhost port

Aucune commande dans le serveur.

Dans le client :
Entrer un message pour l'envoyer à tous les clients connectés au serveur.
Entrer "@nomClient message" pour envoyer un message privé qui sera redirigé à
un client unique.


REPARTITIONS DES FICHIERS :
Les fichiers sont répartis de la façon suivante :

Les 2 jar dans une version "sauvegardée" sont à la racine du projet.

Le fichier pour build le projet (build.xml) est à la racine du projet.

Ce fichier README est à la racine du projet pour plus de facilité d'accès.

Les documents demandés (la RFC, le rapport, le manuel) sont dans le dossier doc.

Les sources sont dans le dossier src.

Les compilations de code sont dans bin et build (build est construit par le
fichier ant).
